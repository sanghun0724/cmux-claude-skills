현재 cmux 워크스페이스를 데이터 분석에 최적화된 2-pane 구조로 설정합니다.

```
+----------------------------------+
| Claude (위 60%)                  |
+----------------------------------+
| Python REPL (아래 40%)           | ← python3 자동 시작
+----------------------------------+
```

## 실행

```
cmux-data
```

## 조건

- cmux 워크스페이스 터미널 안에서 실행해야 합니다.
- `python3`가 PATH에 없으면 REPL 자동 시작을 건너뜁니다.
- 현재 pane이 1개일 때만 실행됩니다.
