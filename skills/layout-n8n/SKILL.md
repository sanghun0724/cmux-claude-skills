현재 cmux 워크스페이스를 n8n 개발 레이아웃으로 설정합니다.

```
+-------------------+----------------------+
| Claude (터미널)   | n8n UI (브라우저)    |
|                   | localhost:<port>     |
+-------------------+----------------------+
```

인자: {{ARGUMENTS}}

## 실행

```
cmux-n8n {{ARGUMENTS}}
```

n8n이 실행 중이지 않으면 먼저 n8n 서버를 시작하세요.

## 조건

- cmux 워크스페이스 터미널 안에서 실행해야 합니다.
- 현재 pane이 1개일 때만 실행됩니다.
