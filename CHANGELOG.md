# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [0.1.0] - 2025-04-19

### Added

- `cmux-organize`: surface 자동 분류 (work / research / tools)
- `cmux-snapshot`: cmux 레이아웃 + Claude 세션 ID JSON 스냅샷
- `cmux-restore`: 스냅샷에서 레이아웃 + Claude 세션 복원 (`CMUX_RESTORE_DELAY` 환경변수)
- `cmux-preview`: 마크다운 실시간 사이드 패널 미리보기
- `cmux-ios`: iOS 개발용 3-pane 레이아웃
- `cmux-n8n`: n8n UI 브라우저 + Claude 레이아웃
- `cmux-data`: Claude + Python REPL 레이아웃
- `cmux-skills`: Claude 스킬 개발 + 파일 탐색 레이아웃
- `cmux-web`: Claude + dev 서버 레이아웃
- `cmux-day-start.example`: 하루 시작 워크스페이스 템플릿
- `hooks/cmux-notify.mjs`: Claude Notification → cmux 알림 브리지
- Claude Code 플러그인 매니페스트 (`cmux-kit`)
- `install.sh` / `uninstall.sh`
